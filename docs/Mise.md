# Mise

## Info

- **mise** (pronounced "**meez**") or "**mise-en-place**" is a **development environment setup tool**.
- Using its **mise.toml** config file, you'll have a consistent way to **setup** and **interact** with your projects no matter what language they're written in.

## Usage

### Install tools

```sh
# Install exact versions from lockfile
mise install
```

### Update tools versions

```sh
# Update tools and lockfile
mise upgrade
```
